História dos computadores

Os primeiros computadores podiam ocupar um andar inteiro, eram feitos com válvulas, foram evoluindo para transistores, circuitos integrados, microprocessadores. Durante a guerra mundial eram utilizados para criptografar mensagens e relizar cálculos rapidamente, podemos destacar o Colossus da Alemanha, Eniac e Mark1 dos EUA.

Início

História da internet

A evolução da internet caminha com a evolução dos computadores, ela serve de modo geral para comunicação. Licklider teorizou uma rede de conexão entre computadores, a partir disso surgem pesquisas de como realizar essa conexão. Em 1969 pela Arpante surge a primeira conexão entre a UCLA e SRI, foi enviada uma palavra mas somente parte dela foi entregue. TCP/IP foi uma criação de Vinton Cerf e Roberr Kahn, para deixar a conexão robusta. Tim Berners-Lee surge para propor a comunicação utilizando hipertexto, cria o primeiro navegador o World Wide Web, em 1991 ele publica a primeira página web. Surgem grandes nomes que marcam essa história como Mark Anderson criador do Netscape, Bill Gates da Microsoft, Steve Jobes da Apple.

Cliente Side

O cliente consome a informação de algum lugar, Internet é o caminho, o Server é onde a informação está armazenada. O cliente pode acessar de diversos dispositivos como celular, tablet, notebook. Algumas informações ficam disponíveis por cache no computador do cliente evitando um carregamento desnecessário. A preocupação da responsividade em difetentes dispositivos, navegadores, fica com o Front-End. O navegador é responsável por interpretar os arquivos, o W3C define as normas e padrões. Quando falamos de aplicação web, dispensa a instalação no dispositivo.

Início

Server Side

Devem ser robustos pois recebem várias requisições ao mesmo tempo. Armazenam arquivos, e-mail, streaming, web. Pode ser simples ou ocupar um andar (Data center). Tipos de servidores:

  • Proxy: antes de acessar a internet, pode bloquear o sitem serve também para autenticação.
  • Firewall: bloquear acesso externo.
  • DNS (Domain Name Server): detém uma lista de nomes e IP, como se fosse o RG

Linguagem de programação Web

Linguagem de programação é um conjunto de instruções e regras para gerar software, meio para chegar em uma solução, uma ferramenta para resolver algo. Linguagens que rodam no lado servidor:

  • PHP
  • Ruby
  • C#
  • JS
  • Java
No lado cliente roda JavaScript, CSS, HTML. Ao acessar o navegador podemos executar a aplicação Web, com o mecanismo que interpreta JavaScript, o V8.
